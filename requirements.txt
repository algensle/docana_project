Standard Libraries Python Version 3.9.25.:

| Package                       | Purpose                             |
| ----------------------------- | ----------------------------------- |
| `re`                          | Regular expressions / text cleaning |
| `json`                        | Reading JSON files                  |
| `csv`                         | CSV export                          |
| `random`                      | Random sentence sampling            |
| `collections` (`defaultdict`) | Transition matrix counting          |


Third Party Packages:
| Package      | Imported As         | Purpose                               |
| ------------ | ------------------- | ------------------------------------- |
| NLTK         | `nltk`              | Frequency distributions, tokenization |
| spaCy        | `spacy`             | POS tagging, tokenization             |
| pandas       | `pd`                | Excel reading, dataframe handling     |
| scikit-learn | `sklearn`           | Cohen's Kappa calculation             |
| matplotlib   | `matplotlib.pyplot` | Bar charts                            |
| NumPy        | `numpy`             | Numerical computations                |
| seaborn      | `seaborn`           | Heatmap visualization                 |


Additional spaCy Model:

| Model            |
| ---------------- |
| `en_core_web_sm` |


Install this by running the following code in your bash:
python -m spacy download en_core_web_sm
